1. The definition of AddTwo(x) is there till the last pass. It is mostly because the function can be called from another files also. The compiler won't understand this before the link time.

2 The function should be declared with keyword Static. When the function is declared with the keyword static,it remains only upto the file where it is declared. The body of the function will vanish if the function is not called in the file. Static sets the scope of function to the containing file.
